# slash-embed

可嵌入的免費工具 widget（供夥伴站 iframe），由 slash-invest.com 提供。
每個 embed 底部含 dofollow attribution 連結回 slash-invest。

- `fee-calc-ar.html` — 阿語交易所手續費返佣計算機（來源產線：`工具/exchange-referral-writer/scripts/embed_calc_ar.py`）

託管：GitHub Pages（main / root）。

- `liq-calc-ar.html` — 阿語合約爆倉/強平價計算機（來源產線同上，config 驅動）
